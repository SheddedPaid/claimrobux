# Claim Your Reward - Discord Nitro & Robux Generator

A modern, interactive web application for claiming Discord Nitro and Robux rewards with a sleek dark theme UI.

## Features

✨ **Modern UI Design**
- Responsive dark theme interface
- Smooth animations and transitions
- Glassmorphism effects with gradient backgrounds
- Mobile-optimized layout

🎮 **Reward Options**
- **Discord Nitro**: Basic and Boost tiers
- **Robux**: Multiple Robux package options (450, 1500, 4500)

⚙️ **Interactive Features**
- Platform selection with custom icons
- Dynamic reward card generation
- Username validation
- Loading progress indicator with smooth animations
- Embedded TapRain offerwall integration

🎨 **Customization**
- Custom SVG and PNG icons
- Gradient color schemes
- Fully responsive design
- Accessible HTML structure

## How It Works

1. Users select a platform (Discord Nitro or Robux)
2. Choose a specific reward package
3. Enter their username
4. Click "Start Generation" to begin
5. Watch the loading animation
6. Complete the TapRain offerwall to claim the reward
